# Kloser 요금제 (Pricing)

Kloser는 **회사 1곳 기준**으로 과금되는 SaaS입니다. 모든 플랜에는 회사를 대표하는 관리자 계정 1개가 포함되며, 플랜에 따라 직원이 사용할 수 있는 **서브 계정 개수**와 **사용 가능한 기능 범위**가 달라집니다.

---

## 핵심 요약

- **과금 단위**: 회사 1곳 기준의 월 단위 비용 (사용자 수가 아닌 회사당 정액)
- **계정 구조**: 관리자 1개(고정) + 플랜별 서브 계정 N개
- **연간 결제**: 일시 결제 시 20% 할인
- **무료 체험 없음**: 대신 도입 전 무료 시연(데모) + 30분 컨설팅 제공

| **연간 결제 할인** | −20% | −20% | 협의 |

## 플랜별 상세

### Starter — 29,000원 / 회사 / 월

1인 사용자가 Kloser의 핵심 가치를 빠르게 체험할 수 있는 입문 플랜.

**계정 구성**
- 관리자 1개 + 직원용 서브 계정 **1개**
- 동시 이용 가능 직원 수: **1명**

**포함 기능**
- 실시간 AI 통화 보조 (월 **200통** 분석 한도)
- 통화 종료 후 자동 요약·메모
- 고객 관리 DB (최대 **1,000명**)
- 이메일 지원 (영업일 기준)

**제한 사항**
- 일일 To-Do 자동 추천 미포함
- AI 뉴스레터 자동화 미포함
- 팀 성과 KPI 대시보드 미포함

**적합한 고객**
- 1인 영업 담당자 / 소규모 사업자 / 1인 기업
- Kloser를 처음 도입해 통화 보조 기능부터 검증하고 싶은 회사
- 통화량이 월 200건 이내인 업무

---

### Pro — 49,000원 / 회사 / 월 ⭐ 가장 많이 선택하는 플랜

1인 사용자에게 Kloser의 모든 기능을 사용량 제한 없이 풀어주는 플랜. 직원 수는 Starter와 동일하지만, 기능이 전부 풀린 것이 차이점입니다.

**계정 구성**
- 관리자 1개 + 직원용 서브 계정 **1개** (Starter와 동일)
- 동시 이용 가능 직원 수: **1명**

**포함 기능**
- **Starter 모든 기능 +**
- **무제한 통화 분석** (월간 한도 없음)
- 일일 To-Do 자동 추천 (네이버 검색 API 기반)
- AI 뉴스레터 자동화 (작성·발송)
- 팀 성과 KPI 대시보드
- 우선 채팅 지원 (응답 시간 1~2시간)

**적합한 고객**
- 1인 사용자이지만 통화량이 많거나(월 200통 초과), 마케팅 자동화·대시보드까지 필요한 솔로 셀러
- Starter를 쓰다가 사용량 한도에 도달했거나 추가 기능이 필요한 회사

---

### Enterprise — 299,000원~ / 회사 / 월 (5인 기준)

5인 영업팀과 보안·규정 준수 요구사항이 있는 조직을 위한 플랜. 5명까지는 정액 299,000원이며, 6명 이상부터는 맞춤 견적입니다.

**가격 구조**
- **5명까지**: 299,000원 / 월 (정액, 부가세 별도)
- **6명 이상**: 맞춤 견적 (직원 수, 부가 옵션, 배포 형태에 따라 협의)
- **결제 주기**: Kloser 담당자가 고객사 현장에 직접 방문해 초기 셋팅을 진행해야 하므로 **연간 구독만 가능** (월간 결제 불가)

**계정 구성**
- 관리자 1개 + 직원용 서브 계정 **5개** (기본)
- 동시 이용 가능 직원 수: **5명** (6명 이상은 별도 협의)
- 다중 조직(자회사·계열사) 관리 옵션은 별도 협의

**포함 기능**
- **Pro 모든 기능 +**
- SSO / SAML 인증 연동 (Okta, Azure AD, Google Workspace 등)
- SLA 99.9% 가동률 보장
- 전담 CS 매니저 + 도입 온보딩 지원
- 온프레미스 / VPC 배포 옵션
- 보안 감사 자료 제공 (ISO 27001 등)
- API 호출 한도 협의

**적합한 고객**
- 영업 담당자 5명 규모의 팀
- 금융·의료·공공 등 보안 규제가 있는 산업
- 자체 인프라(온프레미스/프라이빗 클라우드)에 배포해야 하는 회사
- SSO·감사 로그·SLA가 도입 결정 요인인 조직

> **5명 초과 사용자가 필요하다면?** Enterprise 플랜의 직원 계정 추가 옵션을 협의해 드립니다 — 영업팀에 별도 문의해 주세요.

## 결제 및 할인 정책

| 옵션 | Starter / Pro | Enterprise |
|---|---|---|
| **월간 결제** | 가능 (매월 자동 결제) | **불가** (초기 셋팅으로 연간만) |
| **연간 결제** | 12개월 일시 결제 시 **20% 할인** | **연간 구독만 가능** |
| **2년 약정** | 미제공 | 24개월 약정 시 별도 할인 협의 |
| **결제 수단** | 신용카드 · 계좌이체 · 세금계산서 | 계좌이체 · 세금계산서 (계약서 기반) |
| **신규 가입 혜택** | 도입 전 1:1 무료 시연 + 30분 컨설팅 | 도입 전 1:1 무료 시연 + 도입 컨설팅 + 온보딩 |

> **Enterprise는 연간 구독만 가능합니다.** Kloser 담당자가 고객사 현장에 직접 방문해 초기 셋팅(SSO/SAML 연동, 보안 감사 자료 준비, 고객사 환경 설정, 5인 직원 온보딩)을 진행해야 하므로, 월 단위 해지·재구독 운영이 어렵기 때문입니다.

## 자주 묻는 질문

**Q. 사용자 수가 아니라 회사 기준으로 과금하는 이유는?**
Kloser는 영업 조직 단위로 통화·고객·성과 데이터가 통합되는 도구라, "회사 1곳 = 1 결제 단위"가 자연스럽습니다. 대신 플랜에 따라 직원 수만큼의 서브 계정이 자동으로 발급됩니다.

**Q. Starter와 Pro의 직원 수가 똑같다면 차이가 뭔가요?**
직원 수(서브 계정 1개)는 같지만 **기능 범위가 다릅니다.** Starter는 통화 보조 + 자동 요약 + 고객 관리만 제공하며, Pro는 여기에 무제한 통화·일일 To-Do·뉴스레터·팀 성과 대시보드가 모두 풀립니다.

**Q. 직원이 한 명 더 필요해요. 추가 비용은?**
Starter / Pro에서는 직원 계정 추가가 불가능합니다. 2명 이상이 필요하시면 **Enterprise(5명)** 으로 업그레이드하시거나, 5명을 초과하는 경우 별도 견적을 받으시면 됩니다.

**Q. 도중에 플랜을 변경할 수 있나요?**
네. Starter ↔ Pro 간 업그레이드는 즉시 적용되며 일할 정산됩니다. Pro → Enterprise 업그레이드는 마이그레이션 후 일할 적용됩니다. 다운그레이드는 다음 결제 주기부터 반영됩니다.

**Q. 환불 정책은요?**
구매 후 7영업일 이내에는 전액 환불이 가능합니다. 이후에는 결제 주기 기준으로 미사용 기간을 일할 환불해 드립니다 (Enterprise는 계약서에 따름).

**Q. 무료 체험은 없나요?**
별도의 무료 체험은 운영하지 않습니다. 대신 **도입 전 무료 시연(데모) + 30분 컨설팅**을 통해 회사 환경에 맞는 활용법을 검토하실 수 있습니다.

**Q. 데이터는 어디에 저장되나요?**
Pro까지는 AWS Seoul(ap-northeast-2) 리전에 저장됩니다. Enterprise는 고객사 요청에 따라 온프레미스 또는 별도 VPC 배포가 가능합니다.

## 도입 문의

- **모든 플랜 구매 문의**: 랜딩 페이지 상단 또는 가격 섹션의 "구매 문의" 버튼을 통해 회사 이메일을 남겨주세요. 1영업일 내 담당자가 연락드립니다.
- **Enterprise**의 경우 담당자가 현장 방문 일정과 도입 컨설팅을 함께 안내드립니다.
