01

도입 개요

Kloser는 상담원 보조형 AI 어시스턴트로, 전화망을 직접 대체하지 않고 통화 음성을 분석하여 응대 추천을 제공합니다. 도입 시 통화 환경 표준화가 필요하며, 평균 1~2일이면 첫 통화 분석을 시작할 수 있습니다.

02

사전 준비사항

도입 전 다음 환경이 갖춰져야 합니다. 한 가지라도 충족되지 않으면 정상 동작이 보장되지 않으니 반드시 확인해 주세요.

기업 인터넷전화 가입 필수

Kloser는 PC 기반 인터넷전화의 통화 음성을 캡처하여 분석합니다. 일반 유선전화나 휴대전화만 사용하는 환경에서는 도입할 수 없습니다. KT·LG U+·SK브로드밴드 등 주요 통신사의 기업 인터넷전화 서비스에 가입되어 있어야 합니다.

하드웨어 / 환경

  • Windows 10/11 PC
  • USB 헤드셋 (단일 표준 장비 권장)
  • 안정적 인터넷 회선 (10 Mbps↑)
  • PC 기반 소프트폰 (또는 인터넷전화 앱)

권한 / 정책

  • 마이크 / 시스템 오디오 캡처 권한
  • 통화 녹음·분석 동의 고지 절차
  • 회사 가이드 / FAQ / 응대 매뉴얼
  • 관리자 1인 + 직원 계정 명단
03

단계별 일정

구매 문의부터 첫 통화까지 5단계, 평균 1~2일에 완료됩니다. Enterprise는 현장 방문 셋팅 일정이 추가됩니다.

  1. 1
    D-Day

    구매 문의 & 도입 컨설팅

    홈페이지에서 회사 이메일 등록 → 1영업일 내 담당자 연락 → 30분 도입 컨설팅으로 환경 점검.

  2. 2
    D+1 오전

    계약 & 결제

    플랜 선택, 계약서 서명, 세금계산서/카드 결제 완료. Enterprise는 별도 SOW 작성.

  3. 3
    D+1 오후

    관리자 계정 발급 & 가이드 업로드

    관리자 콘솔에 로그인 → 회사 정보 입력 → 상담 매뉴얼·FAQ 업로드. AI가 자료 학습 시작 (자동 처리, 약 30분).

  4. 4
    D+2 오전

    데스크톱 앱 배포 & 직원 온보딩

    관리자가 직원 서브 계정을 직접 발급 → 직원 PC에 Kloser 데스크톱 앱 설치 → 마이크/시스템 오디오 권한 부여 → 테스트 통화 1회.

  5. 5
    D+2 오후

    실서비스 시작

    평소처럼 인바운드 상담 진행 → AI가 실시간 응대 추천 → 통화 후 자동 메모 생성 → 다음날 아침 일일 To-Do 자동 발송.

자주 묻는 질문

일반 유선전화 / 휴대전화로도 도입할 수 있나요?
현재 버전은 PC 기반 인터넷전화 환경만 지원합니다. 일반 유선전화·휴대전화는 PC에서 통화 음성을 캡처할 수 없어 도입이 어렵습니다. 향후 별도 엔터프라이즈 연동 프로젝트로 PBX/CTI 연동을 검토할 수 있습니다.
통화 내용은 어디에 저장되며, 얼마 동안 보관되나요?
Pro까지는 AWS Seoul(ap-northeast-2) 리전에 저장됩니다. 기본 보관 기간은 90일이며 회사별로 7일~3년 사이에서 설정 가능합니다. Enterprise는 온프레미스 / VPC 배포 옵션이 있습니다.
실시간 응답 지연이 2~3초인 이유는?
음성 캡처(100~300ms 청크) → 스트리밍 STT 전사 → 회사 가이드 RAG 검색 → LLM 추론을 거치며, 안정적인 분석을 위해 발화 단위로 묶어 처리합니다. 이 수준은 동시통역형 응답이 아닌 상담원 보조 용도로 충분합니다.
고객에게 통화 분석 사실을 고지해야 하나요?
네, 통신비밀보호법에 따라 통화 녹취·분석 시 양 당사자 동의 고지가 필요합니다. Kloser는 통화 시작 시 자동 안내 멘트(예: "이 통화는 AI 품질 분석을 위해 녹음됩니다") 기능을 제공합니다.
도입 컨설팅은 무료인가요?
네. 30분 환경 점검 컨설팅과 도입 시연(데모)은 모든 플랜에 무료로 제공됩니다. Enterprise는 추가로 현장 방문 셋팅·온보딩이 포함됩니다.

도입 검토 중이신가요?

회사 환경에 맞는 도입 방안을 30분 무료 컨설팅으로 안내드립니다. 1영업일 내 담당자가 연락드려요.
